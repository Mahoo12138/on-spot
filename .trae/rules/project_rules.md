我要做的一个 App ，是匿名、无社交、纯工具型「现场实时信息公告板」，做现场说说。
用户快速查看 / 发布特定场地的实时情况（排队、人流、失物、现场提醒）合规底线：无用户间互动、无私信 / 联系方式、无个人主页、仅工具属性是一个极简、高效的“去社交化信息工具”，在景区、商场、交通枢纽等高频流动场景非常有杀伤力。

App 使用 supabase 作为后端实现，该项目则是前端，是一个微信小程序项目，使用 typescript + scss 编写：
 + design 目录下是项目的设计稿，包含小程序的页面设计和组件设计。
 + miniprogram 目录下是小程序的代码，包含页面、组件、样式、图片等。
